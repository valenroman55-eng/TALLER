Git

Git es un sistema de control de versiones distribuido, gratuito y de código abierto. En palabras sencillas, es un programa que registra todos los cambios realizados en el código de un proyecto a lo largo del tiempo, permitiendo recuperar versiones anteriores y facilitando el trabajo colaborativo sin riesgo de pérdida de datos

Github

GitHub es una plataforma en la nube utilizada por millones de desarrolladores para almacenar código, llevar un registro de las modificaciones (control de versiones) y colaborar en proyectos de software.
